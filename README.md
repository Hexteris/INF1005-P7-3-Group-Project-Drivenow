# DriveNow

DriveNow is a car rental management system built for both customers and admins. It supports booking cars, managing members, handling payments, tracking loyalty points, referrals, and reviewing booking activity.

## Features

- User registration and login.
- Car browsing and booking.
- Booking management.
- Payment tracking.
- Loyalty points system.
- Referral system.
- Review management.
- Admin dashboard for managing users, cars, bookings, and reports.
